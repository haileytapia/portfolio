## Get started

### Create an account (required)

You must have an account to use Discord.

Your Discord account is now created.

Continue following this guide using the Discord website—the same website on which you created your account—or the [Discord desktop or mobile app](#download-discord-optional).

### Download Discord (optional)

You can use Discord online, but if you want to use it as a desktop or mobile app, follow these steps:

1.  In a web browser, go to [discord.com/download](https://discord.com/download).
2.  Select **Download** for your desktop or mobile device's operating system. The following operating systems are supported:
    *   Windows
    *   Linux
    *   Mac
    *   iOS
    *   Android
3.  (Conditional) Depending on the operating system for which you downloaded Discord in [step two](#step-two), follow these steps:

## User interface

The main components of the interface include:

- **Servers:** These are the communities you can join or create to connect with other users.
- **Channels:** Within servers, you can create or join channels to communicate with other users.
- **Direct Messages:** You can also send direct messages to other users outside of servers.
